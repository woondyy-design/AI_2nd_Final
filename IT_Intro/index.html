SUNJIN LOGISTICS

SUNJIN LOGISTICS는 해상·항공 국제물류주선(Freight Forwarding)을 전문으로 하는 종합물류 기업입니다. 자체 개발한 FMS(Forwarding Management System)를 기반으로, 고객사의 SAP ERP와 표준 인터페이스를 통해 선적요청부터 정산까지의 전 과정을 디지털로 연결합니다.

수출 프로세스

S/R 접수 → 부킹 → Gate In → 선적(Vessel) → B/L 발행 → 출항 → 도착(ETA) → 정산. 각 단계 상태가 FMS로 회신되어 ERP에서도 조회됩니다.

수입 프로세스

사전신고 → 입항(Arrival) → 하선 → 보세운송 → 통관(Customs) → 반출 → Delivery. 통관 상태와 D/O 발행 여부가 실시간 반영됩니다.

부산 신항 vs 북항 수입 마진 비교 계산기를 단일 HTML 파일로 만들어줘. 디자인은 자유롭게 하고, 아래 기능만 정확히 구현해줘.
입력값 (좌측 패널)

매출 청구 단가: USD/톤, 환율 → 톤당 매출 = USD × 환율 자동 표시
물량 정보: 월 물량(톤), 컨테이너당 적재량(톤), 탱크로리 기본중량(톤), 탱크로리 운송단가(원/톤)
매입 원가: 작업료(원/톤·디베닝), 신항 셔틀료(원/대), 북항 셔틀료(원/대)
기타 비용: "+추가" 버튼으로 항목을 동적으로 추가/삭제. 각 항목은 (항목명, 금액, 단위)로 구성하고 단위는 드롭다운으로 원/월(고정), 원/톤, 원/대(컨테이너), 원/대(탱크로리) 중 선택

계산 로직 (신항·북항 각각 동일하게 계산)

컨테이너 수 = ceil(월물량 ÷ 컨테이너당톤)
탱크로리 수 = ceil(월물량 ÷ 탱크로리기본중량)
매출 = 월물량 × USD × 환율
작업료 합계 = 월물량 × 작업료
셔틀료 합계 = 컨테이너수 × 셔틀료 (신항/북항 단가만 다름)
탱크로리 합계 = 탱크로리수 × 탱크로리기본중량 × 운송단가
기타비용 합계: 단위별로 원/월은 금액 그대로, 원/톤은 ×월물량, 원/대(컨테이너)는 ×컨테이너수, 원/대(탱크로리)는 ×탱크로리수
매입 원가 = 작업료 + 셔틀료 + 탱크로리 + 기타비용 합계
월간 마진 = 매출 − 매입원가, 마진율(%) = 마진/매출×100, 톤당 마진 = 마진/월물량, 연간 = 마진×12

출력 (우측, 신항·북항 나란히 비교)

각 항에 대해 컨테이너/탱크로리 대수, 매출, 매입원가 항목별 내역(작업료·셔틀료·탱크로리·기타비용 각 줄), 월간 마진·마진율·톤당 마진·연간 마진 표시
마진이 더 높은 쪽에 "★ Better" 뱃지, 마진이 음수면 카드를 시각적으로 다르게(예: 빨강) 표시
하단에 "○○항이 월 ○○원 더 유리 / 연간 차액 ○○원" 요약

공통 동작

모든 숫자 입력칸에 천 단위 콤마 자동 포맷(커서 위치 유지), 금액은 "원" 단위로 표시
입력값이 바뀔 때마다 실시간 재계산
